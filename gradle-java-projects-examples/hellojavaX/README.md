## hellojavaX是个简单版的JAVA项目结构demo(未使用gradle标准项目结构)

#### 目录结构
```
hellojavaX/
          Main.java
          p1/
             Test1.java
          p2/
             Test2.java
```

## 编译/运行
```
cd hellojavaX/../    # 必须在hellojavaX的外层文件夹中才行!在hellojavaX中否则找不到class...
javac hellojavaX/Main.java    # 必须从外面带上hellojavaX这个目录才行...
java hellojavaX/Main
```
